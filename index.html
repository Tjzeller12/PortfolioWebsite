Projects

    Snake Battle Royale

      Snake Battle Royale is an engaging game where players control a snake, aiming to outlast 60 AI-driven snakes. Each AI snake operates on its neural network, developed using the genetic algorithm to enhance their survival strategies.

    Integral Calculator

      The Integral Calculator is a tool I designed to compute both definite and indefinite integrals. It utilizes a binary search tree to organize mathematical terms by their degree, ensuring efficient computation and user-friendly interaction.

    Monkey Species Classification CNN

      This project involves using a Convolutional Neural Network (CNN) to classify 10 different species of monkeys. I built and trained a custom neural network using TensorFlow, leveraging the "10-monkey-species" dataset from Kaggle. The performance of my model was benchmarked against a pre-trained VGG16 model to gauge its effectiveness.

    Calculator

      In collaboration with three classmates, I developed a multifunctional calculator capable of performing PEDMAS operations (Parentheses, Exponents, Division, Multiplication, Addition, Subtraction). The calculator also allows the user to use the '-' unary operator. This project was implemented in Java using AWT and Swing frameworks, highlighting our focus on creating a robust and user-friendly application.

    Super Mario Baseball

      Super Mario Baseball is a creative project from my Object-Oriented Programming class. It features a hash map to dynamically update stats for various Mario-themed baseball players, display each player's name and statistics, and list the top performers in each statistical category.
